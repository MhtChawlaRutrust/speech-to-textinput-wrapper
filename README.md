# speech-to-textinput-wrapper

Provides S2T support to any Textinput in react native.

## Installation

```sh
npm install speech-to-textinput-wrapper
```

## Usage


```js
import { multiply } from 'speech-to-textinput-wrapper';

// ...

const result = multiply(3, 7);
```
